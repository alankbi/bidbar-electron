Tutorial

Click "Add Script" and input your script title and commands.

When a script is run and finishes executing, a notification is shown.

Click on the notification to view the output of the script.

Click on the up and down arrows to reorder your scripts.

Keyboard Shortcuts

Mac: Click Shift+CMD+ALT+N to run script N (1 ≤ N ≤ 9)

PC: Click Shift+CTRL+ALT+N to run script N (1 ≤ N ≤ 9)

To run script 10, use N = 0.
